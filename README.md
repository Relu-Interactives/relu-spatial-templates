# Relu Spatial Templates

A growing library of **ready-made project templates** for
[Relu Spatial](https://reluspatial.com) — the easiest way to start exploring
the Relu Spatial editor.

Templates are complete starter projects: scenes, assets and interactions
already set up for you, covering experiences like image tracking, face
tracking, VR and 3D. Pick one, make it yours, and publish your own experience
in minutes instead of starting from a blank canvas.

## Using a template

1. Open the **Relu Spatial desktop editor**.
2. Go to **Browse → Project Templates**.
3. Pick a template and click **Use Template**.

The editor sets up a fresh local project from the template — ready to edit,
preview and publish as your own.

## Where are the files?

Template archives are distributed through this repository's
[**Releases**](../../releases), so the file tree here stays empty by design.
You don't need to download anything from this page — the editor handles it
for you.
